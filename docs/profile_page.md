# **Profile Page**

On the profile page users will find information regarding their authorization to External Data Sources, and the function to create API keys for data access.

Authorization to any of the External Data Sources can be obtained or refreshed via the tiles at the top of the page.

Below, an API key can be created to access native MC2DP elements.
