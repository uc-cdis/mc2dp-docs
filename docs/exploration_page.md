# **Exploration Page**

The primary tool for exploring data within MC2DP is the Exploration Page, which offers faceted search of data across data sources. This page can be accessed from the /explorer endpoint or the top navigation bar, by clicking on the “Exploration” icon.

[![Exploration Page Intro][img-exploration_page_intro]](https://mc2dp.data-commons.org/Explorer)
