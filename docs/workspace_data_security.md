# **Data Security and Downloading Results in Workspaces**

Data Security in the Workspaces is managed by the Gen3 microservice [Fence](https://github.com/uc-cdis/fence).

Authorizations managed in the [Profile Page][mc2dp-Platform-Profile] are transferred into a users workspace when it is initiated.
